### 1.2 为什么市场环境如此关键？

```python
# ========== 关键洞察 ==========
"""
在机器人仿真中：
- 环境参数（重力、摩擦力）是固定的
- 策略不需要"知道"自己在什么环境
- 优化的目标是适应这个固定环境

在金融市场中：
- 市场环境（牛市、熊市、震荡）是动态变化的
- 策略必须"识别"当前处于什么环境
- 优化的目标是适应多种环境

例如：
- 趋势跟踪策略在牛市有效，在震荡市失效
- 均值回归策略在震荡市有效，在强趋势失效
- 高波动策略需要降低仓位，低波动策略可以加杠杆

因此，市场环境不是"背景"，而是决策的"核心输入"！
"""
```

## 二、市场环境的维度

### 2.1 市场环境的多层次结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    市场环境的多层次结构                                      │
└─────────────────────────────────────────────────────────────────────────────┘

Level 1: 宏观环境 (Macro Environment)
├─ 经济周期
│   ├─ 扩张期 (Expansion) → 企业盈利增长，股市上涨
│   ├─ 衰退期 (Recession) → 企业盈利下降，股市下跌
│   └─ 复苏期 (Recovery) → 政策刺激，股市反弹
│
├─ 货币政策
│   ├─ 宽松 (Easy) → 流动性充裕，估值提升
│   ├─ 紧缩 (Tight) → 流动性收紧，估值下降
│   └─ 加息/降息 → 影响折现率，影响估值
│
└─ 政治环境
    ├─ 地缘政治 (Geopolitical)
    ├─ 贸易政策 (Trade Policy)
    └─ 监管变化 (Regulation)

Level 2: 市场周期 (Market Cycle)
├─ 牛市 (Bull Market)
│   ├─ 定义: 价格持续上涨，通常 > 20%
│   ├─ 特征: 成交量放大，波动率适中
│   ├─ 策略: 趋势跟踪、动量策略
│   └─ 风险: 盲目追高、泡沫破裂
│
├─ 熊市 (Bear Market)
│   ├─ 定义: 价格持续下跌，通常 > 20%
│   ├─ 特征: 成交量萎缩，恐慌性抛售
│   ├─ 策略: 现金为王、做空策略
│   └─ 风险: 逆势抄底、底部不可测
│
└─ 震荡市 (Sideways/Ranging)
    ├─ 定义: 价格在一定范围内波动
    ├─ 特征: 方向不明确，波动率可能高或低
    ├─ 策略: 均值回归、波段操作
    └─ 风险: 频繁止损、手续费侵蚀

Level 3: 微观结构 (Microstructure)
├─ 波动率Regime
│   ├─ 低波动期 (Low Vol) → 趋势跟踪，加杠杆
│   ├─ 高波动期 (High Vol) → 降低仓位，期权保护
│   └─ 波动率聚集 (Volatility Clustering) → 大波动后跟随大波动
│
├─ 流动性状态
│   ├─ 高流动性 (High Liquidity) → 容易交易，滑点小
│   ├─ 低流动性 (Low Liquidity) → 难以交易，滑点大
│   └─ 流动性枯竭 (Liquidity Dry-up) → 市场崩溃风险
│
└─ 市场情绪 (Sentiment)
    ├─ 贪婪 (Greed) → 可能过度投资
    ├─ 恐惧 (Fear) → 可能过度抛售
    └─ 中性 (Neutral) → 理性决策

Level 4: 资产特性 (Asset Characteristics)
├─ 行业周期 (Sector Cycle)
│   ├─ 科技股: 成长性高，波动率大
│   ├─ 公用事业: 稳定性高，防御性强
│   └─ 周期股: 顺周期波动
│
├─ 市值规模 (Market Cap)
│   ├─ 大盘股: 稳定，机构关注
│   ├─ 中盘股: 成长与稳定平衡
│   └─ 小盘股: 波动大，成长性强
│
└─ 个股特征
    ├─ 动量股: 强者恒强
    ├─ 反转股: 否极泰来
    └─ 波动股: 大起大落
```

```python
# ========== 市场环境状态编码示例 ==========

def encode_market_environment(current_state):
    """
    将市场环境编码为状态向量

    Args:
        current_state: 当前市场状态字典

    Returns:
        environment_encoding: 市场环境编码向量
    """
    encoding = []

    # 1. 宏观环境编码 (one-hot or continuous)
    if current_state['economy'] == 'expansion':
        encoding.extend([1, 0, 0])  # 扩张期
    elif current_state['economy'] == 'recession':
        encoding.extend([0, 1, 0])  # 衰退期
    else:
        encoding.extend([0, 0, 1])  # 复苏期

    # 2. 市场周期编码
    if current_state['market_trend'] == 'bull':
        encoding.extend([1, 0, 0])  # 牛市
    elif current_state['market_trend'] == 'bear':
        encoding.extend([0, 1, 0])  # 熊市
    else:
        encoding.extend([0, 0, 1])  # 震荡市

    # 3. 波动率regime编码
    vol_rank = current_state['volatility_percentile']
    if vol_rank > 0.8:  # 高波动
        encoding.append(1.0)
    elif vol_rank < 0.2:  # 低波动
        encoding.append(-1.0)
    else:  # 中等波动
        encoding.append(0.0)

    # 4. 流动性状态编码
    liquidity_ratio = current_state['volume'] / current_state['avg_volume']
    encoding.append(np.clip(liquidity_ratio, 0.5, 2.0))  # 流动性比率

    # 5. 市场情绪编码
    sentiment = (current_state['advance'] - current_state['decline']) / \
                (current_state['advance'] + current_state['decline'])
    encoding.append(np.clip(sentiment, -1, 1))  # 市场情绪 [-1, 1]

    return np.array(encoding)

# ========== 市场环境-aware的特征工程 ==========

def revise_state_with_environment(raw_state, market_environment):
    """
    考虑市场环境的特征工程

    关键：根据不同市场环境，使用不同的特征
    """
    # 提取基础价格特征
    prices = raw_state[0:20]
    returns = np.diff(prices) / prices[:-1]

    # 识别市场环境
    is_bull = market_environment['trend'] == 'bull'
    is_bear = market_environment['trend'] == 'bear'
    is_sideways = market_environment['trend'] == 'sideways'

    vol_rank = market_environment['volatility_rank']  # [0, 1]

    # 根据市场环境选择特征
    if is_bull:
        # 牛市：趋势跟踪特征
        features = [
            np.mean(returns[-5:]),      # 短期动量
            np.mean(returns[-20:]),     # 长期动量
            prices[-1] / np.mean(prices[-20:]),  # 相对强度
        ]
    elif is_bear:
        # 熊市：风险控制特征
        features = [
            -np.std(returns[-5:]),     # 惩罚高波动
            np.abs(returns[-1]),        # 惩罚大幅波动
            -prices[-1] / np.mean(prices[-20:]),  # 低仓位
        ]
    else:  # is_sideways
        # 震荡市：均值回归特征
        features = [
            -(prices[-1] - np.mean(prices[-20:])),  # 偏离均值
            returns[-1] * returns[-2],  # 反转信号
            np.std(returns[-5:]),      # 波动率
        ]

    # 波动率调整
    if vol_rank > 0.8:  # 高波动期
        features.append(-np.std(returns[-10:]))  # 惩罚波动
        features.append(-1.0)  # 降低仓位信号
    elif vol_rank < 0.2:  # 低波动期
        features.append(np.mean(returns[-5:]))   # 追逐趋势
        features.append(1.0)   # 提高仓位信号

    return np.concatenate([raw_state, features])
```

```python
    reward = -10.0 * abs(position)

    # 震荡市：奖励均值回归
```

import numpy as np

def revise_state_with_environment(raw_state, market_environment):
    \"\"\"
    环境感知的状态表示函数

    Args:
        raw_state: numpy数组，形状(120,)，20天OHLCV数据
        market_environment: dict，市场环境信息

    Returns:
        enhanced_state: numpy数组，原始+计算+环境特征
    \"\"\"
    # 1. 提取基础数据
    close = raw_state[0:20]
    volume = raw_state[80:100]
    returns = np.diff(close) / close[:-1]

    # 2. 识别市场环境
    is_bull = market_environment['market_trend'] == 'bull'
    is_bear = market_environment['market_trend'] == 'bear'
    is_sideways = market_environment['market_trend'] == 'sideways'

    vol_rank = market_environment['volatility_rank']
    sentiment = market_environment['sentiment_score']
    liquidity = market_environment['liquidity_ratio']

    # 3. 根据环境计算特征
    features = []

    # 3.1 趋势识别特征
    ma20 = np.mean(close[-20:])
    ma50 = np.mean(close[-50:]) if len(close) >= 50 else ma20
    trend_indicator = (ma20 - ma50) / ma50  # >0: 牛市, <0: 熊市
    features.append(trend_indicator)

    # 3.2 环境自适应特征
    if is_bull:
        # 牛市：趋势跟踪
        momentum_5d = np.mean(returns[-5:])
        features.append(momentum_5d)
        features.append(ma20 / close[0] - 1)  # 相对强度

    elif is_bear:
        # 熊市：风险控制
        features.append(-np.std(returns[-5:]))  # 惩罚波动
        features.append(-abs(close[-1] / ma20 - 1))  # 惩罚偏离

    else:  # sideways
        # 震荡市：均值回归
        features.append(-(close[-1] - ma20) / ma20)  # 偏离均值
        features.append(returns[-1] * returns[-2])  # 反转

    # 3.3 波动率调整
    if vol_rank > 0.8:
        features.append(-1.0)  # 高波动：降低仓位
    elif vol_rank < 0.2:
        features.append(1.0)   # 低波动：提高仓位
    else:
        features.append(0.0)

    # 3.4 情绪调整
    features.append(np.clip(sentiment, -1, 1))  # 市场情绪

    # 3.5 环境交互特征
    # 动量 × 趋势强度
    trend_strength = market_environment.get('trend_strength', 0.5)
    features.append(momentum_5d * trend_strength)

    # 波动率 × 环境方向
    direction_multiplier = 1 if is_bull else -1
    features.append(np.std(returns[-10:]) * direction_multiplier)

    return np.concatenate([raw_state, features])

def intrinsic_reward_with_environment(enhanced_state, market_environment):
    \"\"\"
    环境感知的内在奖励函数

    Args:
        enhanced_state: revise_state的输出
        market_environment: 市场环境信息

    Returns:
        reward: 标量值，范围[-100, 100]
    \"\"\"
    # 1. 提取环境信息
    is_bull = market_environment['market_trend'] == 'bull'
    vol_rank = market_environment['volatility_rank']
    sentiment = market_environment['sentiment_score']

    # 2. 基础奖励
    reward = 0.0

    # 3. 环境适应性奖励
    if is_bull:
        # 牛市：奖励趋势跟踪
        momentum = enhanced_state[121]  # 假设第121个特征是动量
        reward += 10.0 * momentum

    elif market_environment['market_trend'] == 'bear':
        # 熊市：惩罚风险敞口
        position = enhanced_state[122]  # 假设第122个特征是仓位
        reward -= 10.0 * abs(position)

    else:  # sideways
        # 震荡市：奖励均值回归
        deviation = enhanced_state[123]  # 假设第123个特征是偏离度
        reward += 5.0 * (-abs(deviation))

    # 4. 风险调整
    if vol_rank > 0.8:
        reward -= 5.0  # 高波动：降低奖励
    elif vol_rank < 0.2:
        reward += 2.0  # 低波动：提高奖励

    # 5. 情绪反向
    reward -= 3.0 * sentiment  # 恐慌时加分，贪婪时减分

    # 6. 确保在范围内
    return np.clip(reward, -100, 100)
