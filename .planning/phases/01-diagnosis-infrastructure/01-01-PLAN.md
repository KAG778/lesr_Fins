    ## Step 5: Create exp4.7/diagnosis/feature_quality.py
    Function `compute_feature_quality` implementing DIAG-03.

    ```python
    def compute_feature_quality(states: np.ndarray, rewards: np.ndarray,
                                 original_dim: int = 120) -> dict:
        """Compute quality metrics for LLM-generated features.

        Analyzes features beyond original_dim in the state array.
        Returns dict with:
          - per_feature: list of dicts, each with keys:
              index (int), variance (float), correlation (float, abs spearman),
              p_value (float), information_ratio (float)
          - aggregate: dict with keys:
              mean_abs_correlation (float), max_abs_correlation (float),
              num_degenerate (int), num_significant (int, p < 0.05)
        """
    ```

    For each extra feature column (states[:, original_dim:]):
    - variance: `float(np.var(feat))`
    - If `np.std(feat) > 1e-10`: compute `spearmanr(feat, rewards)` -> (corr, p_val)
    - Else (degenerate): set correlation=0.0, p_value=1.0
    - information_ratio: `float(np.mean(aligned_returns) / np.std(aligned_returns))` where `aligned_returns = rewards * np.sign(corr)`, guard std=0
    - Handle NaN from spearmanr by setting to 0.0
