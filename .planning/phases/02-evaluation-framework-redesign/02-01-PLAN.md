Per D-05 and D-11: Create exp4.9_c/metrics.py with two groups of functions:

PERFORMANCE METRICS (pure numpy, no external deps):
1. sharpe_ratio(returns: list/ndarray, rf=0.0) -> float — annualized Sharpe (mean*252)/(std*sqrt(252)), return 0.0 for <2 returns or zero std
2. sortino_ratio(returns: list/ndarray, rf=0.0) -> float — annualized Sortino using downside deviation (neg returns only), same guards
3. max_drawdown(returns: list/ndarray) -> float — max drawdown as positive percentage, use cumsum + running max
4. calmar_ratio(returns: list/ndarray, rf=0.0) -> float — annualized_return / max_drawdown, return 0.0 if max_drawdown=0
5. win_rate(returns: list/ndarray) -> float — count(positive) / count(non_zero), return 0.0 if no non-zero returns

FACTOR EVALUATION METRICS (per D-11, using scipy.stats.spearmanr):
6. ic(feature_values: ndarray, forward_returns: ndarray, method='spearman') -> float — rank correlation between feature and forward return. Use scipy.stats.spearmanr, return 0.0 for <5 pairs
7. rolling_ic(feature_values: ndarray, forward_returns: ndarray, window=20) -> ndarray — rolling window IC, returns array of length max(0, len(feature_values)-window+1)
8. information_ratio(rolling_ic_series: ndarray) -> float — mean(IC) / std(IC), return 0.0 for std=0 or len<2
9. quantile_spread(feature_values: ndarray, forward_returns: ndarray, n_quantiles=5) -> float — split into n_quantiles groups by feature value, return mean(forward_returns[top_group]) - mean(forward_returns[bottom_group])
