7. rolling_ic(feature_values: ndarray, forward_returns: ndarray, window=20) -> ndarray — rolling window IC, returns array of length max(0, len(feature_values)-window+1)
8. information_ratio(rolling_ic_series: ndarray) -> float — mean(IC) / std(IC), return 0.0 for std=0 or len<2
9. quantile_spread(feature_values: ndarray, forward_returns: ndarray, n_quantiles=5) -> float — split into n_quantiles groups by feature value, return mean(forward_returns[top_group]) - mean(forward_returns[bottom_group])

**STATE VECTOR SLICING CONTRACT (CRITICAL):**

The enhanced state layout is:
  enhanced_state = [raw_state(120) | regime_vector(3) | llm_features(N)]
  indices:          [0:120]          [120:123]           [123:123+N]

  Where N = len(llm_features) from self.revise_state(raw_state).
  N is dynamic — determined by the current LLM-generated feature function.

  There is NO self.feature_extractor attribute and NO self.feature_dim attribute.
  The only way to get LLM features is self.revise_state(raw_state).

**FACTOR METRICS COLLECTION STRATEGY:**

During evaluate(), collect features per step alongside daily returns:

1. In the evaluation loop, for each date where raw_state is available:
   - Call self.revise_state(raw_state) to get llm_features (ndarray of shape (N,))
   - Store llm_features in a list: step_features.append(llm_features)
   - Also store daily returns as before

2. After the loop:
   - Stack step_features into a matrix: feature_matrix = np.array(step_features)  # shape (T, N)
   - Compute forward_returns = daily_returns[1:] (shifted by 1 day)
   - Align: feature_cols use rows [0:len(forward_returns)], i.e., feature_matrix[:-1] if shapes match
   - For each feature dimension j in range(N):
     - ic_val = metrics.ic(feature_matrix[:len(forward_returns), j], forward_returns)
     - rolling = metrics.rolling_ic(feature_matrix[:len(forward_returns), j], forward_returns, window=20)
     - ir_val = metrics.information_ratio(rolling)
     - qs_val = metrics.quantile_spread(feature_matrix[:len(forward_returns), j], forward_returns)
     - factor_metrics[f'feature_{j}'] = {'ic': ic_val, 'ir': ir_val, 'quantile_spread': qs_val}

3. Guard conditions:
   - If revise_state raises or returns all zeros (fallback), set factor_metrics = {}
   - If T < 5 (too few points for meaningful IC), set factor_metrics = {}
   - Wrap the entire factor_metrics block in try/except, set factor_metrics = {} on any error

Replace self._sharpe() and self._max_dd() calls with metrics.sharpe_ratio() and metrics.max_drawdown().
