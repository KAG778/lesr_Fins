## API密钥配置

### 方法1: 使用配置工具 (推荐)

```bash
cd /home/wangmeiyi/AuctionNet/lesr
source /home/wangmeiyi/miniconda3/etc/profile.d/conda.sh
conda activate AuctionNet

python exp4.7/setup_keys.py
```

选择 `2` 使用ChatAnywhere，然后输入你的密钥。

### 方法2: 手动创建配置文件

创建 `exp4.7/.api_keys.txt` 文件：

```bash
# ChatAnywhere配置
OPENAI_API_KEY=sk-your-chatanywhere-key-here
OPENAI_BASE_URL=https://api.chatanywhere.cn/v1
```

或者使用官方OpenAI API：

```bash
# 官方OpenAI配置
OPENAI_API_KEY=sk-your-openai-key-here
# OPENAI_BASE_URL=https://api.openai.com/v1  # 可选，官方API可省略
```

### ChatAnywhere 使用说明

**获取密钥**: 访问 https://chatanywhere.cn/ 注册并获取API密钥

**Base URL**: 
- 主地址: `https://api.chatanywhere.cn/v1`
- 备用: `https://api.chatanywhere.com.cn/v1`

**支持的模型**:
- `gpt-4o-mini` (推荐，性价比高)
- `gpt-4o`
- `gpt-4`
- `gpt-3.5-turbo`

## 实验配置

编辑 `exp4.7/config.yaml`:

```yaml
# LLM配置
llm:
  model: "gpt-4o-mini"  # 推荐使用gpt-4o-mini (性价比高)
  temperature: 0.7
  # ChatAnywhere配置 (取消注释以启用)
  base_url: "https://api.chatanywhere.cn/v1"

# 实验配置
experiment:
  tickers: [TSLA, MSFT]  # 交易股票
  sample_count: 6        # 每轮采样数量 (减少可节省API调用)
  max_iterations: 3      # 迭代次数 (减少可节省API调用)
```

## 成本控制建议

使用ChatAnywhere + gpt-4o-mini可以大幅降低成本：

```yaml
llm:
  model: "gpt-4o-mini"   # 比gpt-4便宜很多
  temperature: 0.7

experiment:
  sample_count: 3        # 减少采样 (默认6)
  max_iterations: 1      # 减少迭代 (默认3)
```

## 常见问题

### Q: 如何验证API密钥是否有效?
A: 运行配置工具选择 `3` 查看当前配置

### Q: 如何跳过优化直接测试已有结果?
A: 使用 `--skip_optimization` 参数:
```bash
python exp4.7/main.py --skip_optimization --results_dir exp4.7/results
```

### Q: API调用失败怎么办?
A: 检查:
1. 密钥是否正确
2. base_url是否可访问
3. 账户是否有余额

### Q: 使用ChatAnywhere的优势?
A: 
- 无需海外信用卡
- 价格更优惠
- 支持所有OpenAI模型
- 国内访问更稳定
