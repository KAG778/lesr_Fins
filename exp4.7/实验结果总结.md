# Exp4.7 LESR 实验结果总结

> 更新时间: 2026-04-08

## 一、实验设置

| 项目 | 配置 |
|------|------|
| **框架** | LESR (LLM-Empowered State Representation for RL) |
| **LLM** | gpt-4o-mini (via ChatAnywhere) |
| **RL算法** | DQN (50 episodes/sample) |
| **原始状态** | 120维量价数据 (20日窗口 × OHLCV + adjusted_close) |
| **LLM优化** | 生成 `revise_state` 和 `intrinsic_reward` 函数 |
| **训练期** | 2018-01-01 ~ 2020-12-31 (3年) |
| **验证期** | 2021-01-01 ~ 2022-12-31 (2年) |
| **测试期** | 2023-01-01 ~ 2023-12-31 (1年) |
| **每轮采样** | 6个LLM生成样本 |
| **迭代数** | 3轮 |
| **初始合格样本** | 至少3个 (init_min_valid=3) |

## 二、四支股票实验结果

### 2.1 测试集总览 (2023全年)

| 股票 | LESR Sharpe | Baseline Sharpe | Sharpe提升 | LESR Return | Baseline Return | LESR MaxDD | Baseline MaxDD | 结果目录 |
|------|------------|----------------|-----------|-------------|----------------|------------|---------------|---------|
| **TSLA** | **1.265** | -0.499 | **显著** | **62.09%** | -10.96% | 35.73% | 21.70% | `results/` |
| **MSFT** | **1.356** | 0.677 | **+100.2%** | **23.73%** | 5.96% | 14.28% | 3.75% | `results/` |
| **AMZN** | **2.316** | 1.938 | **+19.5%** | **72.07%** | 57.15% | 18.46% | 21.83% | `results_amzn_nflx/` |
| **NFLX** | **0.920** | 0.354 | **+159.7%** | **31.37%** | 7.75% | 33.00% | 13.43% | `results_nflx_only_new/` |

**关键发现**: LESR在所有4支股票上均超越Baseline，Sharpe提升幅度从 +19.5% (AMZN) 到显著超越 (TSLA，Baseline为负)。

### 2.2 各股票详细结果

#### TSLA (高波动股)

| 阶段 | Sharpe | MaxDD | Return |
|------|--------|-------|--------|
| 验证期 Best (It1, S1) | 0.289 | 30.15% | 13.37% |
| 测试期 LESR | **1.265** | 35.73% | **62.09%** |
| 测试期 Baseline | -0.499 | 21.70% | -10.96% |

- 最佳策略来自 Iteration 1, Sample 1
- LESR在测试期实现buy-and-hold级收益 (1月买入144, 12月卖出239)
- Baseline在测试期亏损

#### MSFT (稳定股)

| 阶段 | Sharpe | MaxDD | Return |
|------|--------|-------|--------|
| 验证期 Best (It2, S1) | 0.627 | 4.09% | 5.70% |
| 测试期 LESR | **1.356** | 14.28% | **23.73%** |
| 测试期 Baseline | 0.677 | 3.75% | 5.96% |

- 最佳策略来自 Iteration 2, Sample 1
- Sharpe提升 +100.2%，回撤控制相对合理 (14.28% vs 3.75%)
- 验证期表现保守 (Sharpe 0.627)，但测试期大幅改善

#### AMZN (电商巨头)

| 阶段 | Sharpe | MaxDD | Return |
|------|--------|-------|--------|
| 验证期 Best (It1, S5) | 0.905 | 6.52% | 20.99% |
| 测试期 LESR | **2.316** | 18.46% | **72.07%** |
| 测试期 Baseline | 1.938 | 21.83% | 57.15% |

- 最佳策略来自 Iteration 1, Sample 5
- LESR和Baseline测试期表现都很强 (2023年AMZN大涨)
- LESR Sharpe提升 +19.5%，MaxDD更低 (18.46% vs 21.83%)
- 注意: 此实验仅完成2轮迭代 (无Iteration 2)

#### NFLX (流媒体)

| 阶段 | Sharpe | MaxDD | Return |
|------|--------|-------|--------|
| 验证期 Best (It0, S1) | 0.226 | 28.50% | 12.14% |
| 测试期 LESR | **0.920** | 33.00% | **31.37%** |
| 测试期 Baseline | 0.354 | 13.43% | 7.75% |

- 最佳策略来自 Iteration 0, Sample 1 (初始轮即为最优)
- Sharpe提升 +159.7%
- 使用了 init_min_valid=3 机制 (首轮3/6合格)

## 三、迭代过程分析

### 各迭代验证集最佳 Sharpe

| 股票 | Iteration 0 | Iteration 1 | Iteration 2 | 最佳来源 |
|------|------------|------------|------------|---------|
| TSLA | 0.000 | **0.289** | -0.026 | It1 |
| MSFT | 0.507 | 0.345 | **0.627** | It2 |
| AMZN | -0.622 | **0.905** | (未完成) | It1 |
| NFLX | **0.226** | 0.151 | 0.072 | It0 |

### 合格样本率

- 总计生成样本约 70+ 个，大部分验证集Sharpe为负
- 优秀样本占比较低 (约 10-15%)，说明LLM特征工程质量参差不齐
- 验证集最佳样本通常在测试期也表现良好

## 五、待改进方向

1. **样本质量**: LLM生成的特征函数合格率偏低，可考虑改进Prompt或增加验证
2. **迭代效果**: 后续迭代未必优于初始轮 (如NFLX的It0即为最优)，COT反馈机制需优化
3. **回撤控制**: LESR的MaxDD普遍高于Baseline，风险控制有待加强
4. **AMZN实验未完成**: `results_amzn_nflx` 缺少Iteration 2，且NFLX结果以 `results_nflx_only_new` 为准
5. **数据源**: 目前仅用量价数据 (OHLCV)，可考虑引入技术指标或基本面特征
